JavaScript

What is JavaScript? JavaScript is one of the 3 main languages which any web developer should learn, As you have already started learning Java, you should be able to pickup JavaScript quickly.

In this section I will discuss how to code with JavaScript, the fundamentals, and also display some examples of JavaScript.

What does JavaScript do?

  • HTML - The content of a website
  • CSS - Layout and design of the website
  • JS - Change the behavior of the website

Changing content

Changing HTML Content

One of the most useful ways to use JavaScript, is to use the method getElementById()

The point of this method, is it allows you to select any element by its id, for example, say we have a pair of <p></p> tags, which we give an id of "test", we are now able to reference this paragraph using JavaScript, allowing us to change its content.

document.getElementById('example').innerHTML = "Magic!";

Change CSS Styles

  • You can also use JavaScript to change the style of an element, for example changing its colour.
  • This is extremely useful, as it allows for dynamically changing style sheets, rather than a plain page.
document.getElementById("exampleChangeId").style.color = "blue";

Displaying and Hiding content

You are able to use JavaScript to change the display mode of content, which allows for you to hide or show content based on your needs

In the example below, we will make two buttons, one that calls a function to hide, and one that reveals an object

function exampleHide() {
    document.getElementById('kitten').style.display = 'none';
}

function exampleShow() {
    document.getElementById('kitten').style.display = 'block';
}

Running functions

Giving buttons function onClick

To give a button a function is a very simple process, once you have added your custom script to your HTML file with <script src="example.js"></script> in either the <head> or the <body>, all you need to do is add onClick="".

The example below shows how the buttons in the section above were generated.

<button onclick="exampleHide()">Hide</button>
<button onclick="exampleShow()">Show</button>

Loading scripts onLoad

A helpful way you can load scripts is by loading them when something loads, either the script itself, or say the body of a website to make a script run when the script is added, add this line in you .js file

window.document.onload = function();

With the use of this, when the page is loaded, the function will run.

An example of when this would be useful would be for this websites syntax highlighter, which as you can see from the above code snippets, automatically highlights syntax to make the code easier to read and understand. This works by, on the page load, looking for any code blocks, taking the text from them, and figuring out what content to put in a <span> which itself is given a class, which then has CSS applied to it.

JavaScript File

Where to put JavaScript

Okay, I get it, now you know the basics of what you can do with JavaScript (we haven't even scratched the surface of it yet), but one of the most vital things you must be able to do is to know where to place the <script>. With HTML we have three options.

  1. Place a link within the <head>
  2. Place a link within the <body>
  3. Write it within <script> tags in the <body> 
<!DOCTYPE html>
<html>
<body> 

<p id="example">My Paragraph</p>
<button type="button" onclick="function()">Hello</button>

<script>
function function() {
    document.getElementById("example").innerHTML = "Hello World!";
}
</script>

</body>
</html>

Out of these options, there are two you should consider, 1, and 2. Option 3 is valid, however it is advisable to separate your JS from your HTML, just as it is advisable to separate your CSS from your HTML.

You should, normally place your scripts in the <head>, especially if they are lightweight, or if you want them to load before the content of the website, however, if you have a script which may take a second to run, you are best placing it at the bottom of the <body>, this will ensure it is loaded last.

External references

You can also reference scripts outside your local storage, this can be done by specifying the url in the <script> tags.

<script src="https://www.ncl.ac.uk/script.js"></script>

JavaScript Outputs

Output options

With JavaScript you have several different options for output...

  • innerHTML
    • This is a fantastic option for outputting to your HTML, as it can be used to create HTML tags and styling
  • console.log()
    • You can also write to the console from JavaScript, I find this particularly useful for when I am debugging, as Google Chrome developer mode also allows you to explore and debug objects you print to the console.
  • document.write()
    • I suppose this could be used for testing, as it outputs to HTML easily, however if it is run again all of the HTML is deleted on the current page, which in my opinion makes it far less useful.
  • window.alert()
    • Popup windows to display information.

JavaScriptSyntax

Variables

You declare variables in JavaScript like so, with the term var

var y, z; //initialize 2 variables
var x = 21; //initialize x and set it to 21
y = 9; //set y to 9
z = y + x; //set z to y + x (21 + 9)

Literals

JavaScript numbers can be written with or without decimals e.g. 10.50, 1001, Strings are text and are either in double or single quotes e.g. 'Hello World' "Hello World"

Operators

JavaScript has the following operators, +-*/%, which can be used in any combination, and follow standard orders of mathematics

Operator Description
+ Addition
- Subtraction
* Multiplication
** Exponential (X^Y)
/ Division
% Modulus
++ Increment
-- Decrement

JS also has =, which is the assignment operator, to assign values to variables

JavaScript Data Types

Strings

Can use double or single quote brackets

var variable1 = "Hello";
var variable2 = 'Hello';

You can use quotes inside a string, as long as the outside quotes are a different type

var doubleQuotes = "Hello 'boo' ";

Numbers

Unlike Java, JavaScript has one type of number, it can be written with or without decimals also you can use scientific notation for either small or big numbers

var number = 42;
var number = 0.42;
var number = 42e-2;

Arrays

You can make arrays very easily in JavaScript, the array below is accessible through fruit[index] 

var fruit = ["Apple", "Orange", "Pear"]

Objects

An object in JavaScript has properties, and you can declare them as you wish e.g.

var student = {firstName: "Bob", lastName: "Foster", age: 19};
